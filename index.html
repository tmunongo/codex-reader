Codex

A calm, customizable Markdown reader.

Why Codex Exists

Codex was created for readers who work heavily with Markdown-based research. AI deep-research tools produce long Markdown documents. Writers collect notes, excerpts, references, and exported summaries. Traditional editors can display Markdown well, but they always feel like editors first: panes, terminals, and the weight of features you don’t need when you simply want to read.

Codex focuses on clarity and comfort. Open a single file or an entire folder of Markdown documents. Navigate them like a library. Switch themes. Read without the noise of an editor surrounding your work.

Features

  • Open individual Markdown files or whole directories.
  • Clean two-pane layout for effortless navigation.
  • High-quality Markdown rendering for long-form reading.
  • Built-in light and dark themes.
  • Load custom theme files to personalize your reading environment.
  • Lightweight and fully offline.
